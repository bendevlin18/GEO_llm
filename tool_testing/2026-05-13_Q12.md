### Summary & Best Matches

**Direct cortical snRNA-seq hits in humans: ~6–7 datasets** are clearly cortex-focused within this index slice. The full 170K+ index likely contains additional entries not retrieved here, so this should be treated as a **lower-bound estimate**.

#### Strongest datasets:
- **GSE144136** and **GSE213982** are the best-matched pair — both profile **dorsolateral PFC (BA9)** post-mortem tissue using **10x Genomics Chromium**, both have processed `.mtx.gz` files ready for analysis, and they cover MDD in mixed-sex vs. female-only cohorts (34 vs. 38 samples). Excellent for cross-study comparison.
- **GSE174367** is the largest (230 samples) and includes **paired snATAC-seq + snRNA-seq** with `.h5` processed files — ideal for multi-omic cortex analyses in Alzheimer's disease.
- **GSE138852** covers **entorhinal cortex** in Alzheimer's disease (8 donors, 214,000 nuclei) with `.csv.gz` processed data — note this is entorhinal, not prefrontal cortex.

#### Important caveats:
1. **This is not a complete count** — the search returned 101 snRNA-seq hits total, but many are non-human or non-cortical; the human cortex subset shown here represents the clearly matching fraction from this result page.
2. **GSE184950** (substantia nigra, PD) was included for context but is **not cortex**.
3. **GSE241521** has only **n=1 sample**, limiting statistical power.
4. Several datasets (GSE144136, GSE213982) use archived **post-mortem tissue**, which introduces RNA quality considerations typical of brain banks.

## Summary

**~37 snRNA-seq datasets** profile the human cortex in this index. Key highlights:

- **All have processed data available** (✅), typically as `.h5ad`, `.rds`, `.mtx.gz`, or `.csv.gz` files.
- **Cortical regions covered**: dorsolateral prefrontal cortex (dlPFC/BA9/BA46) is the most common (~15 datasets), followed by frontal cortex, entorhinal cortex, temporal cortex, motor cortex, visual cortex, and orbitofrontal cortex.
- **Disease focus**: Alzheimer's disease (~8 datasets), ALS (~3), MDD (~2), Huntington's (~2), Parkinson's (~2), FTD, epilepsy, and bipolar disorder.
- **Development**: Several datasets cover fetal/prenatal neocortex ([GSE214532](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE214532), [GSE217511](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE217511), [GSE204683](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE204683)).
- **Largest cohorts**: [GSE272624](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE272624) (289 samples, ALS), [GSE308462](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE308462) (260 samples), and [GSE254569](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE254569) (92 samples).

> ⚠️ **Caveats**: A few datasets include multi-region or multi-species data where the human cortex is one component (e.g., GSE151761, GSE207334). Some datasets using cortical organoids or iPSC-derived models are included but represent *in vitro* cortex. The kidney/adrenal cortex hits were excluded.
