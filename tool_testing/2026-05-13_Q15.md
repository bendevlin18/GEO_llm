### Summary & Key Notes

**Direct count:** From the full search results (233 methylation datasets + 83 via synonyms), **approximately 12 datasets are explicitly WGBS modality *and* cancer-tagged**. The broader methylation-cancer space includes ~150+ datasets, but most use **methylation arrays (Illumina 450k/27k)**, **RRBS**, or **other_methylation** methods (MBD-seq, MeDIP-seq, HELP-tagging) rather than true WGBS.

**Best matches by use case:**
- **Largest human cancer WGBS:** [GSE243528](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE243528) (n=47, pancreatic cancer, mouse+human organoids) and [GSE52271](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE52271) (n=11, multi-cancer types)
- **Leukemia/hematologic malignancies:** GSE72629, GSE80725, GSE62300, GSE83759
- **Solid tumors:** GSE58695 (colon), GSE71399 (lung/hypoxia), GSE87241 (liver HCC), GSE70090 (lung+liver, includes oxBS)

**Important caveats:**
1. ⚠️ **GSE70090** uses **oxBS-Seq** (oxidative bisulfite), which distinguishes 5mC from 5hmC — methodologically distinct from standard WGBS
2. ⚠️ Several datasets labeled `wgbs` in metadata may use **capture-based bisulfite sequencing** (e.g., SeqCap Epi in GSE71399) rather than true whole-genome coverage — check raw file sizes as a proxy
3. ⚠️ Some large cancer methylation SuperSeries (e.g., GSE52272, n=280) include WGBS as a sub-series (GSE52271) alongside array data
4. 🔬 If your analysis can accommodate RRBS, the cancer-RRBS datasets (GSE83532 n=293 AML, GSE58889 n=149 CLL) offer much larger sample sizes
