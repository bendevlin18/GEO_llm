# CUT&RUN

> 721 datasets | 2015/01/01 – 2026/04/08

Cleavage Under Targets and Release Using Nuclease (CUT&RUN). An antibody-targeted approach that uses tethered MNase to cleave chromatin near bound proteins. Lower background than ChIP-seq, works well with fewer cells. Supplementary files follow the same conventions as ChIP-seq (BED, BigWig).
